## Project Overview

This project demonstrates an end-to-end data analysis workflow starting from web scraping Wikipedia (Forbes Global 2000 ranking) to building an interactive Power BI dashboard.

The goal of the project is to extract, clean, transform, and visualize financial data of Saudi companies listed in the Forbes Global 2000 ranking to uncover insights about performance, sector distribution, and financial strength.

This project reflects practical skills in:

- Web Scraping

- Data Cleaning & Transformation

- Exploratory Data Analysis (EDA)

- Data Modeling (DAX)

- Business Intelligence Visualization
  
---

##  Key Insights

- Saudi Aramco dominates in revenue and market value, significantly outperforming other companies.

- The Banking sector represents the highest number of listed companies.

- Profit margins indicate strong operational efficiency in banking compared to some capital-intensive sectors.

- Market value concentration highlights the economic weight of strategic industries such as Oil & Gas.
